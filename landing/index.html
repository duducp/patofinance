Dúvidas frequentes

Precisa cadastrar cartão ou dados bancários?

Não. O Pato Finance é apenas um assistente de registro. Você insere seus gastos e receitas manualmente ou por comando.

É realmente grátis?

Sim. O bot é 100% gratuito e open source. Veja o código no GitHub.

Meus dados ficam seguros?

Sim. Tudo fica no Supabase PostgreSQL com criptografia. Seus dados são seus — não compartilhamos com terceiros.

Funciona sem internet?

Precisa de internet para o Telegram, mas o bot é leve e funciona até em 3G.

O que fazer depois de iniciar o bot?

Envie seu primeiro comando: /despesa 50 almoço para registrar um gasto, /receita 3000 salário para uma receita, ou /saldo para ver seu saldo. O bot já vem com categorias pré-definidas — não precisa configurar nada.

Preciso criar categorias antes de registrar?

Não. O bot tem categorias pré-definidas (Alimentação, Transporte, Salário, etc.) e cria novas automaticamente se você digitar um nome que não existe. Use /categoria para gerenciar.

Como registro uma despesa com categoria, tags e grupo específicos?

Use os flags opcionais: /despesa 50 --descricao "almoço no shopping" --grupo "Cartão Nubank" --tags #comida #lazer. Para receita: /receita 3000 --descricao "salário" --grupo "Conta Corrente". Se omitir algo, o wizard conversacional guia passo a passo.

Posso editar ou excluir uma transação depois de registrada?

Sim. Use /detalhes ID (o ID aparece no extrato) para ver a transação completa. Lá você encontra botões para editar valor, categoria, data, grupo, tags, descrição ou excluir a transação.

Como funciona o sistema de tags?

Tags são marcadores livres como #ifood, #viagem ou #trabalho. Você adiciona ao registrar com o flag --tags #minhaTag. Depois, use /tag para ver todas com contagem de uso, e clique em qualquer uma para filtrar as transações daquela tag.

Como funcionam as recorrências?

Recorrências criam transações automaticamente todo mês, semana ou período configurado. Use /recorrencia para criar pelo wizard ou transforme uma transação existente em recorrência. O bot roda um cron diário às 06:00 que gera as ocorrências pendentes. Você pode adiantar, pular, arquivar ou editar cada recorrência pelos botões inline.

Como acesso meus dados pelo navegador?

Envie /login no bot para gerar um código de 6 dígitos. Acesse o site do Pato Finance, vá em "Entrar" e digite o código. Pronto — seus dados aparecem no navegador com a mesma conta do Telegram.

Consigo usar no computador?

Sim. O Telegram tem versão Desktop e Web (web.telegram.org). Todos os comandos funcionam igual — você pode registrar, consultar e gerenciar tudo do PC.

O bot não respondeu, o que fazer?

Pode ser instante de lentidão — aguarde alguns segundos. Se não responder, envie /cancelar para limpar qualquer estado pendente e tente novamente. Se o problema persistir, o bot pode estar em manutenção — verifique o status no GitHub.

O que acontece se eu resetar minha conta?

O comando /resetar apaga todas as suas transações, categorias, grupos, tags e recorrências. É uma operação destrutiva — o bot exibe um aviso e você precisa digitar RESETAR para confirmar. Use com cuidado.

Como reportar um bug ou sugerir melhoria?

Abra uma issue no GitHub. O projeto é open source e toda contribuição é bem-vinda. Descreva o problema ou sugestão que a gente analisa.
