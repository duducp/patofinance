Comandos disponíveis

/despesa 50 almoço

Registra uma despesa

/receita 3000 salário

Registra uma receita

/saldo

Saldo do período — use /saldo mes passado, /saldo janeiro

/extrato

Extrato com filtros — use /extrato janeiro 2025, /extrato --grupo Pessoal

/resumo

Resumo por categoria — use /resumo ultimo mes, /resumo --grupo Trabalho

/detalhes 42

Detalhes completos da transação com botões para editar e excluir

/agendadas

Lista transações futuras agendadas — use /futuras como atalho

/buscar mercado

Busca transações por descrição, valor ou tag — use /buscar 150, /buscar #ifood

/grupo

Gerencia grupos (contas bancárias, carteiras)

/categoria

Gerencia categorias (despesa e receita)

/tag

Lista tags com quantidade de transações

/ajuda

Lista todos os comandos. Use /ajuda extrato para detalhes de um comando

/limpar

Remove categorias/grupos sem transações

/resetar

Reseta sua conta (apaga todos os dados)

/login

Gera código de 6 dígitos para acessar o dashboard web

/cancelar

Cancelar operação em andamento

Dúvidas frequentes

Precisa cadastrar cartão ou dados bancários?

Não. O Pato Finance é apenas um assistente de registro. Você insere seus gastos e receitas manualmente ou por comando.

É realmente grátis?

Sim. O bot é 100% gratuito e open source. Veja o código no GitHub.

Meus dados ficam seguros?

Sim. Tudo fica no Supabase PostgreSQL com criptografia. Seus dados são seus — não compartilhamos com terceiros.

Funciona sem internet?

Precisa de internet para o Telegram, mas o bot é leve e funciona até em 3G.
